MathJax 公式编辑学习笔记

为什么用MathJax，我只放一张图：

使用MathJax，公式就可以在页面中这样显示：
\(a^2 + b^2 = c^2\)
公式显示风格完全是可以订制的。具体安装配置见文末。

基本用法

  • 独占一行，居中显示的公式——\[YOURFORMULA\]或$$YOURFORMULA$$。如： \[MYFORMULA+-*/\]
  • 嵌入式的公式——\(YOURFORMULA\)。如 \(MYFORMULA+-*/\)

安装与配置

这里的配置依照的是本博客的搭建需求，原理很简单，自取所需。

  • 最省事的配置方法——把下面的代码放在所需的html的<head>标签中
<script src='https://cdn.mathjax.org/mathjax/latest/MathJax.js?config=TeX-AMS-MML_HTMLorMML'></script>
  • 由于cdn速度感人，博主自行从官方clone下来了MathJax包，放在自己的站内，速度就快多了，所以可以这样——把下面的代码放在所需的html的<head>标签中
<script src='https://zhyack.github.io/mathjax-local/MathJax.js?config=TeX-AMS-MML_HTMLorMML'></script>
  • 当然订制起来也很简单，比如——我的博客是偏暗色的背景，所以可以通过下面的代码将其公式字体设置为亮色——在mathjax的config文件夹下default.js中修改：
	styles: {
			  ".MathJax .merror": {
			    color:   "#FFFFFF",
			    "font-style": "normal",
			    "font-size":  "100%"
			  }
	}
  • 每个页面上加段脚本代码也是可以临时设置的——把下面的代码放在所需的html的<head>标签中
<script type="text/x-mathjax-config">
  MathJax.Hub.Config({
    styles: {
		  ".MathJax .merror": {
		    color:   "#FFFFFF",
		    "font-style": "normal",
		    "font-size":  "100%"
		  }
		
		}
  });
</script>
  • 最后提一下——MarkdownPad2中也可以相应的配置——Tools-Options-Advanced-HTML Head Editor中添加<head>标签，然后预览为html时可以达到同样的效果。从某种意义上来说，确实可以部分取代LaTex了，然而Markdown的\ _ *这几个符号的输入很不友好，给写公式带了一定的影响。
